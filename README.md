# SimulCast — AI 同声传译助手

> 七牛云 × XEngineer 暑期实训营 第三批次 · 题目二

## 📖 项目简介

SimulCast 是一款浏览器端的 AI 同声传译工具，帮助用户实时理解外语演讲、技术分享和国际会议内容。

**核心特性：**
- 🎬 **视频搬运模式**：捕获浏览器标签页的视频+音频，搬到我们的页面上显示
- 🌐 **端到端实时翻译**：使用阿里云百炼 LiveTranslate，语音直接翻译成中文
- ✨ **智能纠错**：自动检测并修正之前识别/翻译的错误
- 🎨 **Babelfish 风格字幕**：毛玻璃效果，清晰不遮挡视频
- ⚡ **低延迟**：音频采集 64ms，端到端延迟约 1 秒

## 🚀 快速开始

### 环境要求

- Python 3.9+
- 阿里云百炼 API Key（[申请地址](https://dashscope.console.aliyun.com/)）

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/muonuo/SimulCast.git
cd SimulCast

# 2. 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # macOS/Linux

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
copy .env.example .env
# 编辑 .env，填入你的 DASHSCOPE_API_KEY

# 5. 启动服务
python main.py
```

### 访问应用

打开浏览器访问 http://127.0.0.1:8000

## 🎯 功能说明

### 1. 视频搬运模式

- 点击"开始"按钮，选择要翻译的浏览器标签页
- 视频画面搬到我们的页面上，字幕叠在视频下方
- 原标签页静音，避免双重声音

### 2. 实时翻译

- 使用阿里云百炼 LiveTranslate 端到端模型
- 支持流式输出，逐字显示翻译结果
- 延迟约 1 秒

### 3. 智能纠错

- 自动检测翻译修正（corrected 类型）
- 修正时字幕闪烁提示
- 只显示当前正在说的句子，不堆叠

### 4. 字幕样式

- Babelfish 风格毛玻璃效果
- 中文 24px 纯白粗体
- 背景半透明，不遮挡视频

## 🧪 测试方式

1. 启动服务器：`python main.py`
2. 打开浏览器访问 http://127.0.0.1:8000
3. 点击"开始翻译"按钮
4. 选择一个有英文音频的标签页（如 YouTube 英文视频）
5. 观察字幕是否正常显示中文翻译
